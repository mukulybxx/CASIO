UIM version, available only when model is decoded
disableSensorDataIntegrity boolean <optional>
 true

When stroke is removed from the model, stroke sensor data will be kept by default, to keep integrity between strokes and sensor data, this flag should be false
unitScaleFactor float <optional>
 1

Conversion factor for strokes points - DIP conversions, when 1, conversion is not needed
transform Generic.Matrix

Ink model transform matrix, could be set on demand
sensorData Array.<InkInput.SensorData>

Ink model sensor input, use setter to register not stroke related sensor data (like hover data for example)
